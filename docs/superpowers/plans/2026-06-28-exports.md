# Exports (pcap comments + usage CSV/JSON) Implementation Plan

> REQUIRED SUB-SKILL: superpowers:executing-plans. Steps use `- [ ]`.

**Goal:** Carry per-packet process attribution into exported pcapng (as comments) and let the Usage tab export the current period as CSV/JSON.

**Architecture:** Extend the existing PcapNGWriter/Reader with `opt_comment` on the Enhanced Packet Block; add a pure `UsageExport` (CSV + JSON) encoder; wire NSSavePanel buttons in PacketsView (comment) and UsageView (export).

**Tech Stack:** Swift 6, Swift Testing, SwiftUI, pcapng.

## Task 1: pcapng packet comments (writer + reader)
**Files:** Modify `Sources/MatrixNetPcap/PcapNG.swift` (CapturedRecord.comment), `Sources/MatrixNetPcap/PcapNGWriter.swift`, `Sources/MatrixNetPcap/PcapNGReader.swift`; Test `Tests/MatrixNetPcapTests/PcapNGCommentTests.swift`
**Produces:** `CapturedRecord(timestampMicros:originalLength:data:comment:)` with `comment: String?` default nil; writer emits `opt_comment` (code 1) + `opt_endofopt`; reader parses it back.

- [ ] Test (RED):
```swift
import Testing
@testable import MatrixNetPcap

@Suite("PcapNG comments")
struct PcapNGCommentTests {
    @Test("a packet comment round-trips through write then read") func roundTrip() throws {
        let writer = PcapNGWriter(linkType: PcapLinkType.ethernet)
        var bytes = writer.header()
        let record = CapturedRecord(timestampMicros: 1_000_000, originalLength: 4,
                                    data: [0xDE, 0xAD, 0xBE, 0xEF], comment: "Safari (pid 42)")
        bytes += writer.packet(record)
        let result = try #require(PcapNGReader.read(bytes))
        #expect(result.records.count == 1)
        #expect(result.records.first?.comment == "Safari (pid 42)")
        #expect(result.records.first?.data == [0xDE, 0xAD, 0xBE, 0xEF])
    }

    @Test("no comment writes a plain block and reads back nil") func noComment() throws {
        let writer = PcapNGWriter(linkType: PcapLinkType.ethernet)
        var bytes = writer.header()
        bytes += writer.packet(CapturedRecord(timestampMicros: 5, originalLength: 2, data: [1, 2]))
        let result = try #require(PcapNGReader.read(bytes))
        #expect(result.records.first?.comment == nil)
        #expect(result.records.first?.data == [1, 2])
    }
}
```
- [ ] Run `swift test --filter "PcapNG comments"` → fails (no `comment` param).
- [ ] Implement:
  - `CapturedRecord`: add `public let comment: String?`; update init with `comment: String? = nil`.
  - Writer `packet(_:)`: build options block when `record.comment` is non-nil/non-empty:
    `optBytes`: u16(1) + u16(UInt16(utf8.count)) + utf8 + pad32, then u16(0)+u16(0) (endofopt). `totalLength = 32 + paddedDataLength + optionsLength` (optionsLength = 0 when no comment). Write data, pad32, then optBytes, then trailing u32(totalLength).
  - Reader `readEnhancedPacket`: compute padded captured length; the options region runs from `dataStart + paddedCaptured` to `blockEnd - 4`. Parse TLV options: on code 1, decode UTF-8 value as comment; stop on code 0. Pass `comment` into CapturedRecord. (Use the block total length at `offset+4` to find blockEnd.)
- [ ] Run → pass. Commit "Add pcapng packet comments (process attribution)".

## Task 3: UsageExport encoder (pure CSV + JSON)
**Files:** Create `Sources/MatrixNetModel/UsageExport.swift`; Test `Tests/MatrixNetModelTests/UsageExportTests.swift`
**Consumes:** `UsageRow`. **Produces:** `enum UsageExport { static func csv(_:) -> String; static func json(_:) -> String }`

- [ ] Test (RED):
```swift
import Foundation
import Testing
@testable import MatrixNetModel

@Suite("UsageExport")
struct UsageExportTests {
    private let rows = [
        UsageRow(periodStart: Date(timeIntervalSince1970: 0), app: "Safari", host: "ex,ample.com",
                 country: "US", bytesIn: 100, bytesOut: 20),
    ]

    @Test("csv has a header and an escaped row") func csv() {
        let csv = UsageExport.csv(rows)
        let lines = csv.split(separator: "\n", omittingEmptySubsequences: false)
        #expect(lines.first == "app,country,host,bytes_in,bytes_out,period_start")
        #expect(lines.dropFirst().first == "Safari,US,\"ex,ample.com\",100,20,1970-01-01T00:00:00Z")
    }

    @Test("empty rows produce just the header") func emptyCSV() {
        #expect(UsageExport.csv([]) == "app,country,host,bytes_in,bytes_out,period_start")
    }

    @Test("json decodes back to the same fields") func json() throws {
        let data = try #require(UsageExport.json(rows).data(using: .utf8))
        let decoded = try JSONSerialization.jsonObject(with: data) as? [[String: Any]]
        #expect(decoded?.count == 1)
        #expect(decoded?.first?["app"] as? String == "Safari")
        #expect(decoded?.first?["bytesIn"] as? Int == 100)
    }
}
```
- [ ] Run → fails.
- [ ] Implement: ISO8601 formatter (UTC); CSV escaping (wrap in quotes when value contains `,`/`"`/newline, double internal quotes); header `app,country,host,bytes_in,bytes_out,period_start`; rows joined by `\n` (no trailing newline). JSON via a `Codable` DTO array with `JSONEncoder` (`.sortedKeys`, `.prettyPrinted`); `bytesIn`/`bytesOut` as Int, `periodStart` ISO string.
- [ ] Run → pass. Commit "Add usage CSV/JSON export encoder".
